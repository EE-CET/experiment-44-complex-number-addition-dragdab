# Experiment 44: Complex Number Addition

## Problem Statement

Create a class `Complex` to represent a complex number with real and imaginary parts.
* Use a constructor to initialize the values.
* Include a method to add two complex numbers and return the result.

## Input Format

* Line 1: Real and Imaginary parts of the first number (space-separated).
* Line 2: Real and Imaginary parts of the second number (space-separated).

## Output Format

* The sum in `a + ib` format.

### Example 1

**Input:**

```text
2 3
4 5
```

**Output:**

```text
6 + 8i
```
